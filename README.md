# Operational Seat Booking Data Analysis (SQL)

## Project Overview
This project focuses on analyzing seat booking and utilization data
generated from a bus seat allocation system to derive meaningful
operational and business insights using SQL.

## Business Objective
- Understand seat utilization patterns
- Identify high-demand routes and travel dates
- Analyze cancellations and seat availability
- Support data-driven scheduling decisions

## Dataset
The dataset used is a sample and anonymized booking dataset created
for portfolio and learning purposes.

## Tools Used
- SQL
- CSV dataset
- Data analysis and reporting concepts

## Key Analysis Performed
- Route-wise booking analysis
- Seat utilization percentage calculation
- Peak booking date identification
- Cancellation trend analysis

## Outcome
The analysis highlights demand patterns, utilization efficiency, and
provides insights that can support better operational planning.

## Notes
This is a self-initiated portfolio project created to demonstrate
SQL-based data analysis skills.
